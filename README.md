# Gaming Autostéréoscopique

## Description
Projet intermédiaire : Affichage 3D d'un cube en rotation qui s'arrête de tourner quand le joueur ferme sa main.

## Installation
### Gitlab
Pour lier un dossier de sa machine au git, utiliser la commande suivante dans le dossier :
```
git remote add origin https://gitlab.telecom-paris.fr/proj104/2025-2026/gaming-as.git
```

### Jeu
Lancer le dossier godot_mapping/godot_cube dans Godot.
